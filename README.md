# Frontend

React + Vite UI for filtering schools.

## Setup

```bash
npm install
npm run dev
```

Runs at Vite's local URL (usually `http://localhost:5173`).

## Scripts

- `npm run dev` - run development server
- `npm run build` - build production bundle
- `npm run preview` - preview production build

## Backend API URL

By default it uses `https://schoolsdataproject-backend.onrender.com`.

To override:

```bash
VITE_API_BASE=http://localhost:3001 npm run dev
```
